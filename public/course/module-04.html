Key prompts for this module Prompt 1 — LLM Concepts for PMs
Explain the following AI concepts to me as a senior PM who needs to make product decisions — not code: (1) What is a context window and how does it affect what my AI feature can do? (2) When should we use RAG versus fine-tuning? (3) What does "temperature" control and why does it matter for user-facing features? (4) When does an AI agent make sense versus a simpler LLM call? Give me a concrete decision rule for each one.
Prompt 2 — Eval Framework
I'm defining success metrics for an AI feature that [describe the feature]. Help me build an evaluation framework covering: (1) 3 output quality metrics we can measure objectively, (2) 2 metrics for user trust and perception, (3) The most likely failure mode and how we'd detect it in production, (4) How we'd catch regression when we update the underlying model. Make each metric as concrete and measurable as possible — no vanity metrics.
